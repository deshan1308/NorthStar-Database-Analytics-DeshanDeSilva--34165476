# NorthStar Databases and Analytics

This repository contains the Colab notebooks used for the Databases and Analytics coursework on the NorthStar Urban Mobility and Logistics case study.

- `01_sql_in_r.ipynb` – SQL queries in R against `northstar.db` to analyse delivery status, duration, complaints, driver performance, and data inconsistencies.
- `02_r_analytics.ipynb` – R analytics and visualisations (delivery duration distribution, duration by driver, route overrides vs duration, customer ratings, and proof-of-delivery analysis).
- `03_python_analysis.ipynb` – Python data processing and analytics using SQLite and pandas (driver performance vs duration, vehicle/incident impact, route overrides and efficiency, and cost efficiency).
- `04_mongodb.ipynb` – MongoDB Atlas development using PyMongo (document design, CRUD operations, aggregation, indexing, query optimisation, and explain plans).

The cleaned datasets were provided via the module materials and are not included in this repository; the notebooks expect them to be available in the Colab working directory when re-run.
